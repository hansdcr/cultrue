## 🚀 使用指南

### 新开发者入门

1. 从 `docs/README.md` 开始
2. 阅读 `project_overview.md` 了解项目
3. 查看 `iterations/README.md` 了解进度
4. 按照 `quick_start.md` 搭建环境
5. 选择当前迭代开始开发

### 日常开发流程

1. 查看 `iterations/README.md` 确认当前迭代
2. 打开对应的迭代文档（如 `iteration-01.md`）
3. 按照任务清单逐项完成
4. 更新任务状态和进度
5. 完成后更新迭代总览

### 计划调整

1. 小调整：直接修改迭代文档
2. 大调整：更新 `iterations/README.md` 表格
3. 新增迭代：创建新文档并更新表格
4. 记录变更原因
