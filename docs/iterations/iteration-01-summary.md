## 🐛 问题和解决

### 问题1: 缺少greenlet依赖

**问题**: 服务关闭时报错 `No module named 'greenlet'`

**原因**: SQLAlchemy异步引擎需要greenlet库

**解决**:
```bash
uv pip install greenlet
```
并更新pyproject.toml添加greenlet>=3.0.0依赖

**状态**: ✅ 已解决
